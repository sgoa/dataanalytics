# Data Analytics Final Report

This project is an analysis of census data comparisons of Bike Vendors by model, frame (carbon or aluminum), and price in various US cities. The graphs drawn with R can be found in the "Figures" folder.

## Getting Started  

These instructions will help you deploy the project on your machine.

### Prerequisites

You will need to download the bikevendors.csv file, and have R inctalled on your machine.

### Installing

Install and load these libraries.
library(gdata)
library(ggplot2)
library(dplyr)
library(tidyverse)
